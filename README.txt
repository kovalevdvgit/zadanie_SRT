
Библиотеки для работ скрипта:

    1.pyautogui
    2.Pillow
    3.OpenCV

Для корректной работы скрипта нужно перейти в директорию где распологается script_cal.py (your_path/zadanie_SRT). 

Важно отметить что в параметрах системы  в дисплее маштаб и разметка доложны быть 100%, если этоот параметр отличается то нарезанные изображения чисел и символов не смогут быть найдены с помощью pyautogui.locateOnScreen 

Скрипт создан и проверен в OS => Windows 10 Pro для рабочих станций(22H2).

Срипт создан под определенный вид графического калькулятора, внешний вид которого приведен на  изображении calculator.png в коне проета!
